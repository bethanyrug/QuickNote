# QuickNote

![QuickNote Banner](https://via.placeholder.com/1200x300?text=QuickNote+App+Banner)

## Overview

QuickNote is a sleek and lightweight note-taking application designed to help you jot down ideas, to-do lists, and reminders on the fly. Built with speed and usability in mind, QuickNote offers a clutter-free interface that lets you focus on your thoughts without distractions.

## Features

- **Minimalist Interface:** Simple, elegant design focused on content.
- **Cross-Platform:** Works seamlessly on Windows, macOS, and Linux.
- **Markdown Support:** Write notes using Markdown syntax for rich formatting.
- **Auto-Save:** Never lose your notes with automatic saving.
- **Search Functionality:** Quickly find notes with keyword search.
- **Keyboard Shortcuts:** Boost productivity with customizable shortcuts.

## Tech Stack

- Electron
- React
- TypeScript
- Styled Components
- SQLite (for local note storage)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bethanyrug/QuickNote.git
   cd QuickNote
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the app locally:

   ```bash
   npm start
   ```

4. To build the application for production:

   ```bash
   npm run build
   ```

## Usage

- Launch the app and start typing to create a new note.
- Use the sidebar to navigate between existing notes.
- Apply Markdown formatting to enrich your notes.
- Utilize the search bar to find notes quickly.
- Customize keyboard shortcuts via Settings.
